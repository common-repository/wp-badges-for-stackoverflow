* Back end settings to put the user id and customize the design.
* Easy to customize
* You can put your reputation section in sidebar

== Installation ==
1. Upload the folder "stackoverflow-badges" to "/wp-content/plugins/"
2. Activate the plugin through the "Plugins" menu in WordPress back end.
3. Go to "Stackoverflow Badges" and put your id and customize your design.
4. use below shortcode to display stackoverflow reputation section in post, page or widget.

`
[stackoverflow-badges] 
`

4. use below shortcode to display stackoverflow reputation section in your template file.

`
<?php  echo do_shortcode('[stackoverflow-badges]'); ?>
`
